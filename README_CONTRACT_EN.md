# 📑 Type of Deployed Contract

## Contract: Advanced ERC20

### 🔧 Main Features
- Based on the OpenZeppelin library
- Solidity version: 0.8.24
- Integrated functions:
  - Mint / Burn
  - Pause / Unpause
  - Whitelist / Blacklist
  - Supply cap (maxSupply)

---

### ⚙️ Compatibility
- Compatible with Foundry and Hardhat
- Multi‑network: Ethereum, Polygon, Arbitrum

---

### 📜 Governance and Security
- Role management: clear separation between admin, minter, burner, and pauser
- Events logged for all critical actions
- OpenZeppelin hooks (_update) enforcing security rules on every transfer
